## 개인정보 처리방침 (Privacy Policy)

본 확장 프로그램(이하 “확장”)은 OTP 자동 입력 기능을 제공하기 위해 다음과 같은 방식으로 사용자 데이터를 처리합니다.

### 1) 수집/접근 대상 데이터
- Microsoft Graph API를 통해 사용자의 최근 이메일 일부(발신자, 본문, 수신 시각)에 일시적으로 접근합니다.
- 접근 목적은 오직 OTP(일회용 인증코드)를 추출하여 로그인 필드에 자동 입력하기 위함입니다.

### 2) 저장 방식 및 기간
- Access Token/Refresh Token은 브라우저의 `chrome.storage.local`에 저장됩니다(암호화는 브라우저/OS 보안 영역에 의존).
- 이메일 내용 자체는 로컬/원격에 별도 저장하지 않으며, OTP 추출 후 즉시 폐기됩니다.
- 로그에는 토큰 만료 시각 등 동작 상태만 남기며, 메일 본문/개인 내용은 기록하지 않습니다.

### 3) 제3자 제공
- 어떠한 개인 데이터도 제3자에게 전송/판매/공유하지 않습니다.

### 4) 권한
- 기능 수행을 위해 Microsoft Graph `Mail.Read` 권한이 필요합니다. 이는 OTP 추출을 위한 최소 권한입니다.

### 5) 사용자 선택권
- 언제든지 확장 프로그램을 제거하거나, Microsoft 계정 연결을 해제할 수 있습니다.
- 토큰 삭제를 원하면 브라우저 확장 데이터(`chrome.storage.local`)를 초기화하거나, 제공되는 로그아웃/연결 해제 기능을 사용할 수 있습니다(해당 UI는 추후 제공될 수 있음).

### 6) 문의
- 개인정보 관련 문의는 리포지토리 이슈를 통해 접수해 주세요.
